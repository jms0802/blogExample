# 📝 블로그 프로젝트

> Express.js, EJS, SQLite 기반의 간단한 블로그 웹 애플리케이션

## 🔧 사용 기술

- **Express.js**: 서버 사이드 웹 프레임워크
- **EJS**: 템플릿 렌더링 엔진
- **SQLite**: 경량화된 관계형 데이터베이스

## 💡 프로젝트 목적

- Express를 활용하여 **RESTful API와 CRUD 로직**을 직접 구현하며 학습
- SQLite를 연동해 **게시글 생성, 편집, 삭제 기능**을 경험
- 서버 사이드 렌더링을 위한 **EJS 템플릿 활용**

## 🚀 배포 주소

- [my-blog-nytail.koyeb.app](https://my-blog-nytail.koyeb.app/)

> Koyeb를 이용하여 배포

## 📌 기능 요약

- 게시글 목록 조회
- 게시글 작성
- 게시글 수정
- 게시글 삭제
